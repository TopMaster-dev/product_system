# Shopify Custom App 作成ガイド（株式会社ロスフォード様向け）

宛先: 株式会社ロスフォード 馬渡様

Shopify Admin API を使用するためのカスタムアプリ作成と
認証情報の取得手順です。所要 10〜15 分程度です。

---

## 必要なもの

- Shopify 管理画面の **オーナーまたは管理者** 権限
- 約 15 分の作業時間

---

## ステップ 1：管理画面のドメイン形式を確認

弊側で必要なのは **`xxx.myshopify.com`** 形式の管理用ドメインです。

1. Shopify 管理画面にログイン
2. 左メニュー → **設定**
3. **ドメイン** をクリック
4. 「Shopify アドレス」欄に表示されている **`xxxxxx.myshopify.com`** をメモしてください

例：`loseford.myshopify.com`

（`www.loseford.com` は顧客向けカスタムドメインで、API では使えません）

---

## ステップ 2：カスタムアプリの開発を有効化

1. 左メニュー → **設定** → **アプリと販売チャネル**
2. 「**アプリと販売チャネル**」画面で、上部の「**アプリ開発**」をクリック
3. 初回のみ「**カスタムアプリの開発を許可**」ボタンが表示されるので、クリックして許可

---

## ステップ 3：カスタムアプリを作成

1. 「**アプリを作成**」ボタンをクリック
2. アプリ名：`Product System Integration`
3. アプリ開発者：ご自身のメールアドレス
4. 「作成」をクリック

---

## ステップ 4：Admin API スコープの設定

アプリの詳細画面が開きます。

1. 「**設定**」タブ → 「**Admin API のアクセススコープ**」セクション
2. 「**設定**」ボタンをクリック
3. 下記の **5 つのスコープ** にチェックを入れてください：

| スコープ名 | 用途 |
|---|---|
| `read_orders` | 注文情報の取得 |
| `read_products` | 商品情報の取得 |
| `read_inventory` | 在庫情報の取得 |
| `read_locations` | 在庫拠点情報の取得 |
| `write_inventory` | 在庫の書き戻し（Phase 1-B で使用） |

4. 「保存」をクリック

---

## ステップ 5：アプリをインストール

1. 上部に「**インストールします**」ボタンが表示されるのでクリック
2. 確認画面で「**インストール**」をクリック

---

## ステップ 6：認証情報の取得

インストール完了後、画面に **2 つの値** が表示されます：

### Admin API アクセストークン

1. 「**API 認証情報**」タブ
2. 「**Admin API のアクセストークン**」セクションに表示される
3. `shpat_xxxxxxxxxxxxxxxxxxxxxxxx` のような形式
4. **このトークンは一度しか表示されません**。必ずコピーして安全な場所に保存してください
5. 万一見失った場合は、アプリを再インストールすると新しいトークンが発行されます

### API キーシークレット（Webhook 検証用）

1. 「**API 認証情報**」タブ
2. 「**API キーシークレット**」セクションに表示される
3. これも `shpss_xxxxxxxxxxxx` のような形式
4. これは何度でも表示できます

---

## ステップ 7：取得した値を Secret Manager に登録

ステップ 1 / 6 で取得した値を、 `docs/03_secret_manager_registration_guide.md` の手順で
GCP Secret Manager に登録してください：

| 値 | 登録先 |
|---|---|
| `xxx.myshopify.com`（ストアドメイン） | 弊側にメール / チャットでお知らせください |
| Admin API アクセストークン | `shopify-access-token` |
| API キーシークレット | `shopify-webhook-secret` |

---

## ステップ 8：Webhook 登録（あとで弊側にて実施）

弊側で Cloud Run のデプロイが完了し次第、下記の Webhook URL を
登録する作業を弊側で実施いたします（馬渡様の作業は不要です）：

| イベント | URL（例） |
|---|---|
| `orders/create` | `https://product-system-xxxxx.a.run.app/webhooks/shopify` |
| `orders/updated` | 同上 |
| `orders/cancelled` | 同上 |

---

## トラブルシューティング

### Q. 「カスタムアプリの開発を許可」ボタンが見当たらない
→ アカウント権限が不足しています。オーナー権限のあるアカウントでログインし直してください。

### Q. インストールボタンを押した後にエラーが出た
→ スコープ選択を忘れている可能性があります。ステップ 4 を確認してください。

### Q. アクセストークンを紛失した
→ アプリ画面で「**アプリをアンインストール**」してから再度インストールしてください。新しいトークンが発行されます。
